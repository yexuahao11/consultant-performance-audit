基于可疑业绩达成率区间与排班休息日联合分析的工具系统。

## 安装与运行

### 前置要求

- Node.js >= 18
- Windows (PowerShell 分析依赖 Windows 环境)
- Microsoft.ACE.OLEDB 12.0 (用于读取 Excel)

### 安装依赖

```bash
cd app
npm install
```

### 启动服务

```bash
# 启动后端 (端口 3000)
npm start

# 启动前端开发服务器 (端口 5173)
npm run dev
```

或使用并发启动：

```bash
npm run dev & npm start
```

### 访问地址

- 前端：http://localhost:5173
- 后端 API：http://localhost:3000

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/analyze2` | POST | 可疑业绩区间分析 |
| `/analyzeSchedule` | POST | 排班休息日异常分析 |
| `/getDetail` | POST | 获取详细报告 |

## 使用说明

### 可疑业绩区间分析

1. 点击"咨询师绩效分析"上传绩效 Excel
2. 点击"咨询师业绩达成率"上传达成率 Excel
3. 设置可疑业绩达成率区间（默认 100%-105%）
4. 点击"开始分析"
5. 查看分析结果摘要和明细表格

### 排班休息日异常分析

1. 点击"咨询师绩效分析"上传绩效 Excel
2. 点击"咨询师排班表"上传排班 Excel
3. 点击"开始分析"
4. 系统自动筛选休息日仍有收费记录的咨询师
